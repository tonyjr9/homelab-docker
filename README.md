# Homelab Docker Stack

A production-ready homelab built with Docker Compose, featuring media management, networking, monitoring, security, and home automation integration. External access is provided through a TP‑Link ER605/OpenVPN. A Home Assistant VM runs on the same Debian host.

- Keep all secrets in local `.env` files (never commit).
- Use generic IPs (e.g., 192.168.0.10X) or `server-ip` in examples.

See README files in each directory for service-specific details.
